# Dimmer
Your screen is dying...
Play as Byte and collect batteries to keep the screen alive!

## Controls:
Arrow Keys/WASD - Move

R - Restart Level

X/Shift - Dash (Uses up power)

Z/Control- Run (Uses up power)

## Tools Used:
Eclipse + Self-Built Java Engine

Audacity

Aseprite

FL Studio 20

ChipTone
